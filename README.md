# COMP2300_Coursework. Sexual_Health_Education_App


## Instructions on running project
This project utilizes Maven to run. Hence some prior modification to the run configuration of your compiler might be needed.
### Recommended compiler - IntelliJ
1. In the Run/ Debug Configuration, add a new configuration and choose Maven.
2. In the command line section of the configuration, add this line "clean javafx:run".
3. Once done, click on apply and ok.
4. You should be able to run the project by pressing on run now.
